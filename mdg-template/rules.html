Tournament Information

Rules & Format

1. Tournament Identity

MDG.gr stands for Mythic+ Dungeon Greece. The tournament is aimed at Greek World of Warcraft players and Greek-speaking community teams.

2. Team Eligibility

  • Each team has 5 main players.
  • Optional substitutes can be allowed by the admin team.
  • Players must use the registered characters unless approved by admins.
  • Community behavior rules apply in Discord, streams, and public chats.

3. Weekly Time Trial Format

  • Each week has selected dungeon(s), affixes, and key level.
  • Teams submit their best valid run before the weekly deadline.
  • Runs must include proof: screenshot, Raider.IO link, Warcraft Logs link, or VOD depending on your final rules.

4. Scoring System

Default template scoring:

  • 1st place: 100 points
  • 2nd place: 90 points
  • 3rd place: 80 points
  • Each lower rank loses 5 points, minimum 10 points.
  • Failed or invalid runs receive 0 points.

5. Tie-breakers

  1. Fastest total time in the week.
  2. Higher key completed.
  3. Fewer deaths.
  4. Admin review, if still tied.

6. Admin Decisions

The admin team can reject runs for rule violations, missing proof, character swaps, exploits, or unsportsmanlike behavior.
